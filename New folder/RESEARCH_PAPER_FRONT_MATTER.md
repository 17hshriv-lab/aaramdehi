## Abstract

Aaramdehi is a secure and user-friendly e-commerce web application built with React, Vite, TailwindCSS, Express, and Firebase Realtime Database. It supports product browsing, search, comparison, wishlist, cart management, checkout, and basic administration. This report describes the project objectives, system design, implementation details, testing results, and potential future enhancements.
